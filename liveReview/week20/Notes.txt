Map Interface ----> does NOT extend Collection Interface, but we can consider as part of Collection Framework
-holds pairs of Data: key - value
-keys must be unique (not duplicated)
- values can be duplicated
- does not support primitives

HashMap(C) implements Map Interfaces
-order is random
-faster
-accepts null key
-using hash code

LinkedHashMap(C)
-maintains the insertion order
-accepts null key

TreeMap(C)
- it is sorted by the keys
- key cannot be null
